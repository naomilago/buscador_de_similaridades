FAQ

1. Quais tecnologias foram utilizadas?

     Para a construção desta ferramenta, foram utilizadas as linguagens Python e JavaScript, bem como também as biliotecas Gensim, NLTK, Pandas, Numpy e Sklearn do Python. Para o deploy, foi utilizada a hospedagem Hostinger.

2. Qual a licença deste projeto?

     Este projeto é distribuido sob a licença Creative Commons Zero v1.0 Universal (CC0 1.0).

3. Como contribuir com o projeto?

     Para contribuir, basta realizar um fork no repositório no Github. Como sugestão de próximos passos, tenho como objetivo melhorar a interface e também adicionar o percentual de similaridades.
